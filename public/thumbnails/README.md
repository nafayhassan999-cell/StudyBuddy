## How to Get Thumbnails:

### Option 1: Download from YouTube
1. Open the playlist URL in your browser
2. Right-click on the first video thumbnail
3. Select "Save image as..." or use browser DevTools to download the image
4. Rename to the corresponding filename above

### Option 2: Use YouTube Thumbnail URL
For any YouTube video, you can get the thumbnail using this pattern:
```
https://img.youtube.com/vi/[VIDEO_ID]/hqdefault.jpg
```

### Option 3: Create Custom Thumbnails
You can create custom 480x360 (or 1280x720 for HD) thumbnails that match your course branding.

Recommended dimensions:
- Standard: 480x360px
- High Quality: 1280x720px (16:9 aspect ratio)

## Fallback:
If thumbnails are not available, the app will still function. You can use placeholder images or the courses will display without thumbnails.
